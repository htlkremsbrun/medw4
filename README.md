## ue03-read-ajax

In der `tabledata.php` Username und Passwort setzen:

````php
$user = "";
$pwd = "";
````

In der `tabledata.js` URL ergänzen:

````javascript
url: 'http://127.0.0.1/.../tabledata.php'
````
